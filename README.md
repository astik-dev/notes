# Notes
**Notes** is a web application for creating, editing, and managing notes.

## Features

### Note Styling
- Customize text alignment
- Change text color
- Apply text formatting: **bold**, *italic*, and underline
- Adjust text size and line height
- Enable horizontal scrolling for long content
    
### Additional Functionality
- Popup confirmations when closing or refreshing unsaved notes
- Automatically opens the last accessed note on page load
- Autosave notes when switching between notes or creating a new one
- Set default style settings for new notes
- Delete all notes at once or reset all settings to default
